# 書籍管理システム additional_answer

このフォルダは、SPA入門 総合演習の追加課題まで実装した解答例です。

## 追加実装している機能

- 価格の昇順・降順による並び替え
- 登録画面の画像プレビュー
- 登録・更新時の入力チェック強化
- 登録・更新・削除後の画面内メッセージ表示
- トップページの登録書籍数・最近登録された書籍表示
- ログイン機能
- `role` による権限管理

## ログイン情報

| 権限 | ログインID | パスワード |
|---|---|---|
| 管理者 | `admin` | `admin123` |
| 一般ユーザー | `user` | `user123` |

管理者は登録、更新、削除を利用できます。
一般ユーザーは一覧、検索、詳細表示だけ利用できます。
